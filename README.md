# Ros2AutoSetup (ROS2 自动安装脚本程序)
A shell script program to auto install ROS2 in Ubuntu server, this script only install for humble version, if you need install jazzy version, you only need replace humble to jazzy in script file, but pls make
sure your ubuntu server version is 22.04 for humble version and 24.04 for jazzy version.
